# Aquarium Water Log

A small, offline-friendly tracker for aquarium water parameters — pH, ammonia (NH₃),
nitrite (NO₂), nitrate (NO₃) and water changes. Built as a single `index.html` file with
no dependencies, designed to be used on a phone (add it to your home screen).

**Features**
- One-tap entry with quick-value chips and live colour-coding (safe / caution / danger)
- Nitrogen-cycle trend charts and a "latest reading" status card
- Optional free-text notes per reading
- CSV export (for Excel / Sheets) and JSON backup / restore

**Your data stays in your browser.** Readings are saved locally on your own device via
`localStorage` — nothing is uploaded, synced, or stored in this repository. The file here
only contains default seed data; your own readings never leave your phone. Use the in-app
**Backup** button now and then to keep an off-device copy.

## Use it
Open `index.html` (or the hosted GitHub Pages URL) in your browser, then add it to your
home screen to use it like an app.
